this proyected was generated by node version 22.14.0 with the framework Express 5.1.0


## Depedencies packages

Install the follow dependencies packages:
```
npm install express ## Version -> ^2.8.5

npm install dotenv ## Version ->  ^17.2.1

npm install cors ## Version ->  ^12.8.5

npm install mssql  ## Version -> ^11.0.1

npm install multer ## Version -> ^2.0.2

 
```

## Development server 

To start a local development server run:
```
npn run dev (nodemon) or npm run index.js (estatic)
```
